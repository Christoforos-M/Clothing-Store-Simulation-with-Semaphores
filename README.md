# Προσομοίωση Καταστήματος Ενδυμάτων με Σημαφόρους (Java)

Το project αυτό προσομοιώνει τη λειτουργία ενός καταστήματος πώλησης ενδυμάτων με χρήση πολυνηματικής επεξεργασίας και σημαφόρων σε Java.  
Στόχος είναι η υλοποίηση συγχρονισμού και αμοιβαίου αποκλεισμού κάτω από συγκεκριμένους περιορισμούς χωρητικότητας.

Κάθε πελάτης εισέρχεται στο κατάστημα, δοκιμάζει ρούχα σε δοκιμαστήρια ανά φύλο και στη συνέχεια κατευθύνεται στο ταμείο.

## 📚 Πληροφορίες
- Μάθημα: ΠΛΗ 47
- Υποεργασία: Προσομοίωση Καταστήματος με Συγχρονισμό
- Γλώσσα: Java

## 📝 Περιγραφή

Η προσομοίωση ακολουθεί τους παρακάτω κανόνες:

- Στο κατάστημα μπορούν να βρίσκονται ταυτόχρονα το πολύ **40 πελάτες**.
- Υπάρχουν δύο χώροι δοκιμής:
  - Ένας για γυναίκες
  - Ένας για άντρες  
  Κάθε χώρος διαθέτει **5 ξεχωριστά δοκιμαστήρια**.
- Υπάρχει **ένα ταμείο**, ενώ λόγω περιορισμών COVID δεν επιτρέπεται να βρίσκονται στην ουρά περισσότερα από **10 άτομα**.
- Η κατανομή πελατών θεωρείται **50% άντρες / 50% γυναίκες**.
- Κάθε πελάτης παραμένει στο δοκιμαστήριο για **3–10 δευτερόλεπτα** πριν πάει στο ταμείο.
- Ο χρόνος εξυπηρέτησης στο ταμείο είναι σταθερός (π.χ. **5 δευτερόλεπτα**).
- Ένας νέος πελάτης εισέρχεται στο κατάστημα κάθε **2–5 δευτερόλεπτα**.

Όλα τα χρονικά διαστήματα μπορούν να τροποποιηθούν για δοκιμές σε μικρότερο χρόνο.

Οι σημαφόροι χρησιμοποιούνται για να διασφαλίσουν:

- Τον μέγιστο αριθμό πελατών στο κατάστημα
- Τη διαθεσιμότητα δοκιμαστηρίων ανά φύλο
- Το όριο αναμονής στο ταμείο
- Τον αμοιβαίο αποκλεισμό όπου απαιτείται

Κάθε πελάτης υλοποιείται ως ξεχωριστό νήμα.
